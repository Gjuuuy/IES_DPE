# IES_DPE

Projet de prédiction de la consommation énergétique des logements et
d’estimation du Diagnostic de Performance Énergétique (DPE) avant et
après travaux de rénovation.

## Problématique

Comment prédire la consommation énergétique d’un logement selon ses
caractéristiques et estimer l’évolution de son DPE après des travaux
de rénovation énergétique ?

## Approche

Deux modèles complémentaires sont développés :

- Modèle 1 : Régression pour prédire la consommation énergétique
- Modèle 2 : Classification pour prédire la classe DPE

Ces modèles sont intégrés dans une interface utilisateur permettant
de simuler différents scénarios de rénovation.

## Structure du projet

- analyse/ : exploration et compréhension des données
- science/ : développement des modèles
- viz/ : visualisation et dashboard
- frontend/ : interface utilisateur (GitHub Pages)
- data/ : données brutes et transformées
- docs/ : documentation méthodologique

## Branches Git

- main : version stable et déployée
- analyse : data analysis
- science : data science
- viz : data visualisation

## Déploiement

L’interface utilisateur est publiée via GitHub Pages à partir de la
branche main.

## Auteurs
