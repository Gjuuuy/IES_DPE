# Front-end

Ce dossier contient l’interface utilisateur finale.

## Objectif

Permettre à un utilisateur de :
- saisir les caractéristiques d’un logement
- simuler des travaux de rénovation
- visualiser la consommation et le DPE estimés

## Déploiement

Le fichier index.html est utilisé comme point d’entrée
pour GitHub Pages.
